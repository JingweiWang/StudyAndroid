# *TCP* 连接池

在此项目中, 实现了两个 TCP 连接池.

用于作为 TCP 服务器接收和发送数据.

## 示例1. *oldtcp*

*TCP_PORT* 为 *60000*

这个写的比较垃圾(非常垃圾).

### 发送格式:

```json
{
    "from": "123456",
    "content": "content"
}
```

### 存在问题:

1. 每次发送的时候都要带着 *form* , 否则服务器不知道是谁发的.
2. *TCPConnectionPool* 维护连接池时, 操作冗余, 恶心.

## 示例2. *tcp*

对上一个版本进行了优化, 解决了上一个版本的所有问题.

*TCP_PORT* 为 *60000*

### 发送格式:

第一次(加入连接池):

```json
{
    "from": "123456",
    "content": "HI"
}
```

以后:

```json
{
    "content": "content"
}
```
